Description

This project analyzes the 2021 T-Mobile breach as a cybersecurity case study. It summarizes what happened, what information was exposed, how people were affected, and what actions followed.

It also highlights lessons about protecting sensitive data, monitoring systems, and improving security practices to reduce future risk.

Skills Learned

  • Security incident analysis
  • Summarizing technical events clearly
  • Understanding user impact and organizational risk
  • Research writing and citing sources
  • Deriving best-practice takeaways
