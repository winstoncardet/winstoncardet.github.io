Description

This project focuses on designing and testing affordable, customizable 3D-printed prosthetic limbs. The goal is to improve accessibility for people in underserved communities by using 3D printing to reduce cost and speed up production while keeping the prosthetic lightweight and functional.

The plan includes researching existing 3D-printed prosthetic designs, modeling new designs in CAD (Fusion 360), printing prototypes using materials like PLA/TPU, and evaluating strength, weight, flexibility, comfort, and usability.

Skills Learned

  • Research and literature review to inform engineering decisions
  • CAD modeling workflow (Fusion 360) for iterative design
  • Prototyping and material testing (PLA/TPU) for strength and comfort
  • User feedback collection and improving a design based on usability
  • Project planning using a timeline and budget constraints
